Приклад з тегом <article>

HTML5 представив кілька нових семантичних елементів, які роблять ваш код більш змістовним і зрозумілим. Один із таких елементів — <article>.

Використовуйте <article> для представлення автономного контенту, який може існувати окремо, наприклад, блог-пост, новину чи запис на форумі.

Цікаві факти

  • HTML5 був випущений у жовтні 2014 року.
  • Семантичні елементи полегшують розуміння коду.
  • Елемент <aside> ідеально підходить для бічних панелей.
Що таке HTML?

HTML (HyperText Markup Language) — це мова розмітки, яка використовується для створення веб-сторінок і веб-додатків.

Як працює тег <details>?

Цей тег дозволяє створювати розгортувані блоки контенту. Користувач може натиснути на заголовок (вказаний у <summary>), щоб побачити прихований контент.

Що таке тег <summary>?

Тег <summary> використовується для створення заголовка, який відображається перед розгортанням вмісту. Якщо <summary> не вказано, весь контент буде прихованим за замовчуванням.
